Кодировка файлов исходного кода: windows-1251 / CP1251

Чтобы открыть doxygen документацию, откройте в браузере файл "html\index.html"

Piece - базовый абстрактный класс шахматной фигуры, от которого наследуются все остальные фигуры
Position - стандартный класс, хранящий координаты на шахматной доске (от 0 до 7 по вертикали и горизонтали)

file - вертикаль (a b c d e f g h на шахматной доске) типа char
rank - горизонталь (1..8 на шахматной доске) типа int
